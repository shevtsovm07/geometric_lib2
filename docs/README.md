# Geometric Library - Документация

## Общее описание
Библиотека для вычисления площади и периметра геометрических фигур.

## Функции круга

### area(r)
Вычисляет площадь круга.

Пример: area(3) → 78.54

### perimeter(r)
Вычисляет длину окружности.

Пример: perimeter(5) → 31.42

## Функции квадрата

### area(a)
Вычисляет площадь квадрата.

Пример: area(4) → 16

### perimeter(a)
Вычисляет периметр квадрата.

Пример: perimeter(4) → 16
